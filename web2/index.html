TIPOS DE ETIQUETAS

En HTML y CSS, tenemos diferentes tipos de etiquetas, cada una de las cuales tiene un propósito específico. La etiqueta se puede componer de dos formas.

Estructura de Etiquetas
Etiquetas que solo se apertura Etiquetas que tienen una apertura y un sierre
Este tipo de etiqueta común mente se utilizan para indicar una dirección u origen se componen por argumento u opciones tales como src href. Este tipo de etiquetas común mente se utilizan para encerrar un contenido tales como texto img entre otros

Parrafos

La etiqueta que se usa para crear parrafos en HTML es la etiqueta <p> Si se quieren hacer varios parrafos separados se deve de usar la misma etiqueta varias veces

  • <p> </p>----------->Esta etiqueta sirve para crear parrafos

ETIQUETAS DIRIGIDAS A OBJETOS

  • Estructuras

      Estas etiquetas tambien son conocidas como cajas o de tipo objeto ya que podemos configurar su ancho y alto

    • <header> </header>

      • |-Este es utilizado para definir un encabezado de página, normalmente se coloca en su interior la etiqueta <nav> </nav>.

    • <nav> </nav>

      • |-Esta etiqueta es está dirigida al menú de navegación del sitio web normalmente se encuentra dentro de la etiqueta header.

    • <section> </section>

      • |-Los sitios web están por múltiples páginas y cada página está compuesta por un header y un número indefinido de secciones en las que se tratan diferentes temas.

    • <aside> </aside>

      • |-Esta etiqueta se utiliza muy poco, se implementa cuando se desea agregar un apartado a tu sitio web como la publicidad lateral de las paginas comerciales.

    • <footer> </footer>

      • |-Esta etiqueta es utilizada para colocar un pie de pagina

Propiedades CSS

Propiedades generales

  • background-color

    • |-Esta etiqueta sirve para poder colorear las cajas de las etiquetas de caja

  • Padding

    • Esta etiqueta es el espacio entre el contenido del elemento y su borde

  • Width

    • |-Esta etiqueta marca la anchura de las etiquetas de caja imagen

  • Height

    • |-Esta etiqueta marca la altura de las etiquetas de caja imagen

  • marning

    • Esta etiqueta establece los márgenes de un elemento especificando entre uno y cuatro valores, donde cada valor es una longitud, un porcentaje o auto.

Propiedades para texto

  • font-family

    • |-Esta etiqueta es usada para designarle un tipo de letra o tipo de fuente a los textos o titulos de nuestra pagina webp

  • font-size

    • |-Esta etiqueta se utiliza para cambiarle el tamaño de letra o tamaño de fuente a los textos o titulos de nuestra pagina

  • text-aling

    • |-Esta etiqueta es utilizada para la posición en la que queremos que vaya un texto

Propiedades para cajas

  • border

    • |-Esta etiqueta es utiliza para insertarle un borde a una etiqueta de pagina

  • Width

    • |-Esta etiqueta marca la anchura de las etiquetas de caja imagen

  • Height

    • |-Esta etiqueta marca la altura de las etiquetas de cada imagen

¿Comó se conecta un css a un html?

La respuesta a esta pregunta es muy sencilla ya que lo que se debe de hacer es que en la etiqueta head de coloca la etiqueta link en la cual luego se debe de agregar lo siguiente rel="stylesheet" href="css/css.css". En la etiqueta de href se coloca el nombre de la carpeta donde se encuentra tu archivo seguido del nombre que le diste a tu archivo
